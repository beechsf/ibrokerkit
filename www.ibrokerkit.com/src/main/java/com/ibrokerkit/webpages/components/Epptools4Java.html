This component builds directly on NeuStarâ€™s EPP toolkit. It provides the following functionality on top of the toolkit:

  • Maintaining a connection pool and automatic re-opening of connections.
  • Convenience methods to facilitate EPP operations such as registrations, updates, transfers, renewals, etc.
  • Compatibility methods for converting between OpenXRI data structures and EPP toolkit data structures, e.g. for service endpoints, Refs, etc.
  • Storage of every single EPP transaction. This makes the i-broker compliant with section 3.10.2.1 of the I-Broker Agreement of the Global Service Specification at http://gss.xdi.org/.
