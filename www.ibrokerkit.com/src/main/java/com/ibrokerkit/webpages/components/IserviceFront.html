Description

iserviceFront is an implementation of the three i-services described at http://iss.xdi.org/. These i-services are mandatory for every i-broker.

  • OpenID service
  • Contact Page service
  • Forwarding service

iserviceFront contains the web applications that provide the three i-services.

Packaging

iserviceFront is packaged as a .WAR file for deployment at i-service endpoints, e.g. authn.youribroker.com, contact.youribroker.com and forwarding.youribroker.com.
