Description

openxriAdmin is an optional web application that allows administrators to manage data from the openxri component.

This is the unmodified version from the open source project at http://www.openxri.org/.

A public instance of this component is deployed at http://admin.testxri.com.

Packaging

openxri.admin is packaged as a .WAR file for deployment, e.g. admin-openxri.youribroker.com.
