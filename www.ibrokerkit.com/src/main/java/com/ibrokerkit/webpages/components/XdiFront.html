Description

xdiFront is an experimental XDI endpoint for i-brokers. It provides external applications with the following functionality:

  • Check if an i-name's password is correct.
  • Retrieve an i-name's i-number and synonyms (anyone can do this).
  • Retrieve an i-name's private and public keys (only the i-name owner can do this).
  • Store, retrieve and share arbitrary XDI data associated with the i-number.

See Examples: xdiFront for some information on how to talk to an xdiFront endpoint.

About XDI

For more information about XDI, see:

  • XDI on Wikipedia
  • OASIS XDI TC

Packaging

xdiFront is packaged as a .WAR file for deployment at an XDI endpoint, e.g. xdi.youribroker.com.
